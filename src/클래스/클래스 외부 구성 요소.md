
# 패키지와 임포트

  ## 패키지
    비슷한 목적으로 생성된 클래스 파일들을 한곳에 모아 둔 폴더

  ## 임포트
    다른 패키지 내의 클래스를 사용하기 위한 문법 요소
    
    방법1: 클랫스의 풀네임 사용 '패키지명.클래스명'
    방법2: 임포트 사용 'import.패키지명.클래스명'

  ## * 기호로 임포트할 때 주의할 점
     bin - pack1 - pack2   ----- C.class  
                 - A.class     - D.class
                 - B.class

    다음과 같은 폴더 구조로 되어 있다고 가정했을 때 import pack1.*; 하면 A.class, B.class만 임포트 되고 하위 폴더에 존재
    하는 C.class, D.class는 임포트 되지 않는다/


# 외부 클래스
    1개의 소스파일 안에는 최대 1개의 public 클래스만 존재할 수 있고 그 클래스명은 파일명과 동일해야 한다.
    따라서 외부 클래스는 public을 붙일 수 없고 따라서 다른 패키지에서 import 할 수 없다.
